[Title] [Numpy] Cannot mix numpy scalar and MXNet numpy ndarray

Mixing the original numpy array and the mxnet numpy array will trigger some errors:

Minimal reproducible example:
```python
import mxnet as mx
import numpy as np
mx.npx.set_np()
a = np.array(1)
a += mx.np.array(1)
```
