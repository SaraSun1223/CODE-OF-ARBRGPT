[Title] tf.string.format is not returning Unicode characters

As also described in [this Stackoverflow post](https://stackoverflow.com/questions/59795552/tf-string-format-is-not-returning-unicode-characters), when a string passes through `tf.strings.format` unicode characters are not represented correctly.

```python
import tensorflow as tf

print(tf.constant('ğŸ˜Š:ğŸ˜Š').numpy().decode('utf-8')) # output: ğŸ˜Š:ğŸ˜Š
print(tf.strings.format("ğŸ˜Š:{}", tf.constant('ğŸ˜Š')).numpy().decode('utf-8')) # output: ğŸ˜Š:"\\360\\237\\230\\212" 
```

Error observed in tensorflow 2.2 and 2.3
