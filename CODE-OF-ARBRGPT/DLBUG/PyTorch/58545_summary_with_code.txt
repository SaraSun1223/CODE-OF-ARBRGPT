[Title] torch.randperm given GPU random generator raise error

## üêõ Bug

<!-- torch.randperm raise RuntimeError when I set both default tensor type and Generator device to GPU -->

## To Reproduce

Steps to reproduce the behavior:

```
import torch
rng = torch.Generator(device="cuda")
torch.set_default_tensor_type('torch.cuda.FloatTensor')
torch.randint(low=0, high=10, size=(1,), generator=rng) # Works well
idx_pts = torch.randperm(n=50, generator=rng) # Fail and raise error
```

and will get an error:
`RuntimeError: Expected a 'cuda:0' generator device but found 'cuda'`

and even if I enforce the device to cuda, error still remains:
`idx_pts = torch.randperm(n=50, device='cuda', generator=rng)`

## Expected behavior

No error should raise, just like other random functions
