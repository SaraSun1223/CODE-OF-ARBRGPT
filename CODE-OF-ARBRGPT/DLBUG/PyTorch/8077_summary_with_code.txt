[Title] [Bug] Parameters turn into Tensors after load

## Issue description

After a model is saved and then loaded using torch.save / load method, an attribute that was a `Parameter` becomes a `Tensor`

## Code example
```

import torch

class M(torch.nn.Module):
    def __init__(self):
        super().__init__()
        self.B = torch.nn.Parameter(torch.Tensor())

class A(torch.nn.Module):
    def __init__(self):
        super().__init__()
        self.m = M()

a = A()
print(type(a.m.B))
torch.save(a, 'test')
a = torch.load('test')
print(type(a.m.B))
```
The output is
```
<class 'torch.nn.parameter.Parameter'>
<class 'torch.Tensor'>
```
